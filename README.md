# SC2002 Project: University Career Services Portal

A Command Line Interface (CLI) application designed to simulate a university career services portal. This system connects students with employment opportunities while providing administrative tools for staff and system administrators. The project heavily emphasizes Object-Oriented Programming (OOP) design and structural best practices.

## 🌟 Core Features

* **Role-Based Access Control (RBAC):** Designed a backend architecture handling distinct permissions for Students, Staff, and Admins.
* **Scalable Architecture:** Applied SOLID principles to create scalable user profiles and job listing architectures. 
* **Robust Data Persistence:** Managed data persistence using CSV file handling to ensure reliable local storage without the overhead of an external database.

## 🛠️ Technology Stack
* **Language:** Java
* **Architecture:** Object-Oriented Programming (OOP)
* **Storage:** Local CSV File I/O
* **Interface:** Command Line Interface (CLI)
